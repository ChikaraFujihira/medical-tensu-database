# rule-tables

社会保険支払基金が提供する電子点数表をもとにした、診療報酬の**機械的ルール**を格納するディレクトリです。

## 含まれるデータ

- 併算定不可テーブル
- 診療行為コード間の関係（Relation）
- 加算・特定点数の前提条件
- その他機械的に扱える算定ロジック

## 例：フォルダ構成

rule-tables/
└── R06/
├── cannot_combine.csv # 併算定不可
├── add_on_requirements.csv
└── INFO.md


## 用途

AI算定結果の「安全弁」として利用します。  
自然言語の推論結果に対し、機械的ルールで最終チェックを行うことで誤算定を防止します。
