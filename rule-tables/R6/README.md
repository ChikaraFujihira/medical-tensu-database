# rule-tables R6

R6（令和6年度）診療報酬改定に対応した「電子点数表（支払基金）」由来の算定ルールを配置するディレクトリ。

- `supplemental/` : 01補助マスターテーブル（補助マスター）の正規化データ
- `inclusive/`    : 02包括テーブル（包含・被包含ルール）
- `exclusive/`    : 03-1〜03-4 背反テーブル（併算定不可ルール）
- `inpatient-base/` : 04入院基本料テーブル（入院基本料に関する条件）
- `count-limits/` : 05算定回数テーブル（同一日・同一月などの算定回数制限）

central-master/R6 の診療行為マスターと組み合わせて、
AI算定エンジンのロジック判定に利用する。
