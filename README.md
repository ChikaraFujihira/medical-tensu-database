# medical-tensu-database

医科診療報酬に関するデータを体系的に整理し、AI による算定コード生成の基盤として活用するためのオープンデータベースです。

このリポジトリは以下の 3 層構造でデータを管理します。

- **central-master/**  
  厚生労働省が提供する診療報酬データ（中央マスター）を格納します。

- **rule-tables/**  
  社会保険支払基金が提供する電子点数表をもとにした、併算定不可・加算前提などの機械的ルールを格納します。

- **natural-language-rules/**  
  診療報酬改定解説書（通称しろぼん）や疑義解釈など、算定要件の自然言語ルールを OCR/WIP として格納します。

本データベースは、AI による算定判断・診療行為コード生成に必要な「根拠データ」の整備を目的としています。
