# central-master

厚生労働省が提供している診療報酬・医薬品・特定保険医療材料等の「中央マスター」を格納するディレクトリです。

## 含まれるデータ

- 診療行為コード
- 名称
- 点数
- 分類（診療区分）
- 薬剤・材料等のコード体系
- 年度ごとの更新データ

## フォルダ構成（例）

central-master/
└── R06/ # 令和6年度 診療報酬改定版
├── medical.csv
├── drug.csv
├── material.csv
└── INFO.md # データ元URLなど


## 用途

AI算定の「公式基準データ」として利用します。本データは制度そのものの仕様を表すため、今後の改定にも最優先で対応する必要があります。
