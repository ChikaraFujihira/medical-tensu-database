# Natural Language Rules (R6)

このディレクトリは、R6 年度の「自然言語ベースの診療報酬算定ルール」を構造化して管理するための領域です。

しろぼん、疑義解釈、告示・通知など、中央マスターや電子点数表では表現できない自然言語要素をまとめています。  
AI による算定判断では、rule-tables（電子点数表）の機械的ルールと本ディレクトリの自然言語ルールを統合して利用します。

---

## 構成

R6/
├─ raw/ # 原典資料（要約・自作メモのみ）
├─ normalized/ # AIに渡すための構造化済データ(JSONLなど)
├─ schema/ # JSONスキーマ
└─ README.md


---

## データ種別

### ● raw/
自然言語資料の“自作要約だけ”を置きます。  
しろぼん原文の保存は禁止です（著作権保護のため）。

### ● normalized/
- rules.jsonl：しろぼん・通知の算定ルール要約（構造化）
- qanda.jsonl：疑義解釈の差分ルール（構造化）
- patterns.json：自然言語 → ロジック変換パターン辞書

### ● schema/
normalized データの JSON スキーマ。

---

## 運用方針

1. しろぼん／疑義解釈を読み、要点を raw/ にメモ  
2. 要点を構造化し normalized/ の JSONL に追加  
3. rule-tables と組み合わせて算定ロジックに活用  
4. 改定ごとに R8/、R10/ を並列で用意して年次更新

---

## 注意事項

- しろぼん全文・疑義解釈全文の保存は禁止  
- 本ディレクトリには要約・自作メモのみ格納  
- 年度を跨ぐデータは原則混在させない
