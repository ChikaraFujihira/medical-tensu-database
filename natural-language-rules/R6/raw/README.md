# raw/ — 原典資料（要約・メモ保管領域）

このフォルダには、しろぼん・疑義解釈・告示などの自然言語資料の  
**要約（自作メモ）** を格納します。

著作権の都合上、原文の全文保存は禁止です。

---

## 構成

raw/
├─ shirobon/ # しろぼんや書籍由来の要約
└─ q_and_a/ # 改定後の疑義解釈PDFのメモ


---

## shirobon/
しろぼんの該当箇所を読み、以下の形式で要約して格納します：

- どの章・節か（ページ番号）
- 算定要件
- 算定不可条件
- 例外・特記事項
- 注意点

---

## q_and_a/
疑義解釈（PDF）を参照し、自作のまとめとして以下を記録します：

- 問番号
- 質問の要点
- 回答の要点
- 該当するコード
- 元ルールへの影響

---

## 運用メモ

- 原文のスクリーンショットを置くことも禁止  
- 必ず“自分の言葉でまとめた文章”のみ  
- JSON化は normalized/ で行う
